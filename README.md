# Automatic Flashcard creation for Anki using chatGPT
Utilises gpt4 to create beautiful flashcards from csv export from [Language reactor](https://www.languagereactor.com), including synonyms, definitions, grammar analysis and more. 

- Generates Anki cards with essential fields like cloze deletions, hints, examples, and formatted definitions.
- Adds furigana (reading annotations) to Japanese words for easier pronunciation learning.
- Handles errors gracefully, preventing excessive retries and saving problematic entries for manual review.
- Outputs the generated Anki deck as an .apkg file for import into the Anki software.
- Provides options for customization through a configuration JSON file.
